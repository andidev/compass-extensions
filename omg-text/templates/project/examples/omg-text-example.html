OMG Text Example

The omg-text extension provides a mixin with a simple syntax for creating your awesome text shadows. There are three shadow styles for you choose from, flat, shaded, and offset, below are examples of each. Please visit the github page for detailed mixin descriptions and more usage examples.

Fantastically Flat
mixin: @include omg-text(hsla(163, 25, 45, 1), 135, 35, flat)
change angle
  1. 0°
  2. 45°
  3. 90°
  4. 135°
  5. 180°
  6. 225°
  7. 270°
  8. 315°
Seductively Shaded
mixin: @include omg-text(hsla(310, 60, 75, 1), 225, 20, shaded, 5)
change angle
  1. 0°
  2. 45°
  3. 90°
  4. 135°
  5. 180°
  6. 225°
  7. 270°
  8. 315°
Orderly Offset
mixin: include omg-offset(180, #008ce2, 4, #055990, 7)
change angle
  1. 0°
  2. 45°
  3. 90°
  4. 135°
  5. 180°
  6. 225°
  7. 270°
  8. 315°

Installing extension

Getting started is easy, just install the gem from your terminal.

(sudo) gem install omg-text

Now that you have omg-text installed, you can create a project based on the omg-text extension or install it into an existing project

Creating a new project

compass create <project_name> -r omg-text --using omg-text

#import into your scss/sass file
import "omg-text"

Adding to an existing project

# Edit the project configuration file and add:
require omg-text

# From command line
cd your/project/folder
compass install omg-text

#import into scss/sass file
import "omg-text"
